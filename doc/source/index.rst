Preface
=======

`slice_lid` is a package to train NOvA SliceLID classifiers.

If you are new to `slice_lid` it might be worth to get started by going through
the tutorial :doc:`tutorials/mprod5/index`.

`slice_lid` borrows significant amount of code from the `lstm_ee`_ package.
Most `lstm_ee` manuals are equally applicable to the `slice_lid`.

.. _lstm_ee:  https://github.com/usert5432/lstm_ee
